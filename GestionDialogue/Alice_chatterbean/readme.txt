Commande pour lancer le chatbot de type ALICE :
java -cp bsh.jar -jar chatterbean.jar ./Bots/properties.xml


Fichier de contexte :
Bots/context.xml

Fichiers AIML :
Bots/Mati/

java -jar bsh.jar console.bsh - lancement en mode console;
java -jar bsh.jar gui.bsh - lancement avec interface graphique
